# 验收记录

本记录用于说明当前版本已经完成并通过的主要验收项。

## 构建与依赖

- pnpm build 通过。
- 使用 pnpm 作为包管理工具。
- 未生成 package-lock.json。
- 未生成 yarn.lock。

## 数据导入与字段处理

- Excel 导入通过，支持 `.xlsx` 和 `.xls`。
- 默认读取第一个 Sheet。
- 首行字段名识别通过。
- 字段别名自动映射通过。
- 缺字段空状态通过。
- 缺字段不会阻断导入。
- 缺字段时不编造假数据。
- 异常数据不会导致页面崩溃，会生成数据质量提示。

## 看板与报告

- 数据看板动态刷新通过。
- 核心指标计算通过。
- 年龄、学历、任职年限、绩效图表生成通过。
- 人才九宫格和继任风险预警通过。
- 分析报告生成通过。
- 报告内容复用当前看板和导入数据。

## 导出与打印

- PPT 导出通过。
- Word 导出通过。
- PDF 导出通过。
- Excel 报表导出通过。
- Excel 模板下载通过。
- 打印样式通过，打印时会隐藏顶部按钮、Tab、导入控件和导出菜单。

## Windows 启动器

- Windows 一键启动器通过。
- 可检查 Node.js 和 pnpm。
- 可在首次启动时自动安装依赖。
- 可启动本地开发服务。
- 可在服务就绪后自动打开浏览器。
- 新增一键环境部署并启动脚本，可在缺少 pnpm 时通过 corepack 准备 pnpm。
- 新增本地 `.runtime` 机制，可在缺少 Node.js 时下载免安装版 Node.js。

## 持久化

- localStorage 持久化通过。
- 刷新页面后，最近一次导入数据和生成报告可以恢复。
- 清空当前数据后，localStorage 缓存会清除。

## 已知提示

- Vite 构建时可能提示部分 chunk 体积较大，原因是 PPT、Word、PDF、Excel 导出库体积较大。该 warning 不影响系统构建、启动和运行。
