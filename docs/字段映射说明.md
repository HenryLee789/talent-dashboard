# 字段映射说明

系统会读取 Excel 第一个 Sheet 的第一行作为字段名，并自动映射为标准字段。字段名会先去除前后空格、合并连续空格、去除末尾冒号，并忽略大小写差异。

如果同一个标准字段被多个表头命中，系统只使用第一个命中的字段，其余重复字段会标记为未映射。

## 缺字段对分析的影响

| 缺少字段 | 可能影响 |
| --- | --- |
| 性别 | 男/女性别比 |
| 年龄、出生日期 | 平均年龄、35 岁及以下占比、年龄分布 |
| 学历 | 硕士及以上占比、学历分布 |
| 入职日期、任职年限 | 平均任职年限、任职年限分布 |
| 绩效等级 | 绩效表现图 |
| 是否关键人才 | 关键人才总数 |
| 人才分类 | 人才九宫格 |
| 是否有继任者、继任风险 | 继任风险 |

## 字段缺失时的原则

- 缺字段不会阻断导入。
- 系统不会编造假数据。
- 可由其他字段推导的指标会尽量推导，例如通过出生日期推导年龄、通过入职日期推导任职年限。
- 无法推导的指标、图表或报告段落会显示空状态或缺字段说明。
- 导出文件会复用当前系统里的缺字段提示。
