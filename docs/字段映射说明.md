# 字段映射说明

关键人才管理数据看板系统会读取 Excel 花名册第一个 Sheet 的第一行作为字段名，并根据内置字段字典执行字段映射。字段匹配前会去除首尾空格、合并连续空格、去除末尾冒号，并忽略大小写差异。

同一个标准字段被多个表头命中时，系统使用第一个命中的表头，其余重复表头会标记为未映射。

## 缺失字段影响范围

| 缺失字段 | 影响范围 |
| --- | --- |
| 性别 | 男/女性别比 |
| 年龄、出生日期 | 平均年龄、35 岁及以下占比、年龄分布；提供其中任一字段时，相关分析可继续计算 |
| 学历 | 硕士及以上占比、学历分布 |
| 入职日期、任职年限 | 平均任职年限、任职年限分布；提供其中任一字段时，相关分析可继续计算 |
| 绩效等级 | 绩效分布图 |
| 是否关键人才 | 关键人才总数 |
| 人才分类 | 人才九宫格 |
| 是否有继任者、继任风险 | 继任风险 |

## 处理原则

- 缺失字段不会阻断 Excel 花名册导入。
- 系统不会生成虚构数据。
- 可由其他字段推导的指标会继续计算，例如由 `出生日期` 推导年龄、由 `入职日期` 推导任职年限。
- 无法计算的指标、图表或报告段落会显示空状态或缺失字段说明。
- 导出文件会复用当前系统中的缺失字段说明和数据质量提示。
