# 关键人才管理数据看板系统使用说明

本文档面向关键人才管理数据看板系统的最终使用者，说明 Windows 环境下的启动、Excel 花名册导入、数据看板查看、分析报告生成和导出流程。

## 1. 启动系统

下载仓库 ZIP 并解压后，在项目根目录双击：

```text
START_HERE.bat
```

该脚本会检查本地运行环境，自动准备 Node.js、pnpm 和项目依赖，并在应用启动后打开浏览器。

中文入口同样可用：

```text
一键部署环境并启动.bat
```

已配置 Node.js 和 pnpm 的环境，也可直接使用：

```text
启动人才看板.bat
```

应用启动后，浏览器访问地址为：

```text
http://127.0.0.1:5173/
```

关闭启动器窗口后，本地服务会停止。

## 2. 下载 Excel 花名册模板

进入应用后，点击页面右上角“下载模板”。系统会下载：

```text
关键人才花名册模板.xlsx
```

模板包含：

- `花名册`：填写正式导入数据。
- `填写示例`：提供字段填写参考，正式导入前应移除示例数据。
- `填写说明`：说明字段规则和支持的字段别名。

## 3. 填写花名册

在 `花名册` Sheet 中填写员工数据。

填写规则：

- 第一行必须为字段名。
- 字段名前不要增加标题、说明或空行。
- 每一行代表一名员工记录。
- 示例数据仅用于参考，不应作为正式数据导入。
- 真实员工花名册和导出报告不应提交到公开仓库。

## 4. 导入 Excel

点击页面右上角“导入Excel”，选择填写完成的 `.xlsx` 或 `.xls` 文件。

导入完成后，可在“数据录入”Tab 查看：

- 导入数据行数。
- 字段映射结果。
- 缺失字段。
- 数据质量提示。

## 5. 查看数据看板

进入“数据看板”Tab，查看系统基于 Excel 花名册生成的核心指标和分析模块：

- 关键人才总数。
- 男/女性别比。
- 平均年龄。
- 硕士及以上占比。
- 35 岁及以下占比。
- 平均任职年限。
- 年龄、学历、任职年限和绩效分布。
- 人才九宫格和继任风险预警。

## 6. 生成分析报告

点击“生成报告”，选择“生成当前月报告”。

系统会基于当前 Excel 花名册生成分析报告，内容包括总体概览、人才结构分析、绩效分析、风险提示、管理洞察和行动建议。

未导入 Excel 花名册时，系统会显示示例数据和示例报告，用于展示页面结构。

## 7. 导出报告

点击“生成报告”下拉菜单，可导出：

- PPT 报告。
- Word 报告。
- PDF 报告。
- Excel 报表。

如导出前尚未生成分析报告，系统会先生成当前月分析报告，再导出对应文件。

## 8. 打印

点击“打印”可打印当前页面内容。

打印时，系统会隐藏顶部按钮、Tab、导入控件和导出菜单，保留适合打印的正文内容。

打印内容取决于当前 Tab：

- “数据看板”Tab 打印数据看板。
- “分析报告”Tab 打印分析报告。
- “数据录入”Tab 打印数据录入摘要。

## 9. 清空数据

点击“清空当前数据”后，系统会清除浏览器 localStorage 中保存的导入数据、字段映射结果、缺失字段、数据质量提示和已生成分析报告。

清空操作只影响当前浏览器中的本地缓存，不会删除本机上的 Excel 文件。

## 10. 常见问题

### Windows 启动器无法启动

优先使用 `START_HERE.bat`。该脚本会自动准备本地运行环境。环境部署过程中下载失败时，检查网络连接后重新运行脚本。

### 部分指标或图表为空

Excel 花名册缺少对应字段，或对应字段没有可用数据。可在“数据录入”Tab 查看字段映射结果、缺失字段和数据质量提示。

### 字段名称不完全一致是否可以导入

可以。系统支持常见中文别名和英文别名。完整字段清单见 [字段映射说明.md](字段映射说明.md)。

### 真实员工数据如何处理

关键人才管理数据看板系统为本地运行的前端工具。真实员工花名册、包含敏感个人信息的 Excel 文件和导出的分析报告不应提交到公开仓库。
