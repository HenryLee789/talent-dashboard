# 使用说明

本文档写给系统使用人员，不需要了解代码。

## 第一步：启动系统

在项目文件夹里找到：

```text
启动人才看板.bat
```

双击这个文件即可启动系统。

第一次启动时，系统可能会自动安装依赖，需要等待几分钟。请不要关闭弹出的命令行窗口。

启动成功后，系统会自动打开浏览器。如果浏览器没有自动打开，可以手动访问：

```text
http://127.0.0.1:5173/
```

如果启动失败，请检查电脑是否已经安装 Node.js 和 pnpm。

停止系统的方法：

```text
关闭启动器弹出的命令行窗口即可。
```

创建桌面快捷方式的方法：

1. 右键 `启动人才看板.bat`。
2. 选择“发送到 > 桌面快捷方式”。
3. 以后双击桌面图标即可启动。

## 第二步：下载 Excel 模板

进入系统后，点击右上角“下载模板”按钮。系统会下载一份 Excel 模板：

```text
关键人才花名册模板.xlsx
```

模板里有三个 Sheet：

- `花名册`：正式导入数据应填写在这里。
- `填写示例`：只用于参考，不要作为正式数据保留。
- `填写说明`：说明字段填写规则和支持的字段别名。

## 第三步：填写花名册

打开模板，在“花名册”Sheet 中填写员工数据。

注意事项：

- 第一行必须是字段名。
- 不要在第一行前面增加标题、说明或空行。
- 每一行代表一名员工。
- “填写示例”Sheet 只是参考，正式导入前请使用“花名册”Sheet 填写数据。
- 请不要把真实敏感员工数据提交到公开 GitHub 仓库。

## 第四步：导入 Excel

回到系统页面，点击右上角“导入Excel”按钮，选择填写好的 `.xlsx` 或 `.xls` 文件。

导入后可以进入“数据录入”Tab 查看：

- 导入了多少条数据。
- 字段映射结果。
- 缺少哪些字段。
- 数据质量提示。

## 第五步：查看看板

点击“数据看板”Tab，查看系统自动生成的关键人才管理看板，包括：

- 关键人才总数。
- 男/女性别比。
- 平均年龄。
- 硕士及以上占比。
- 35 岁及以下占比。
- 平均任职年限。
- 年龄、学历、任职年限、绩效图表。
- 人才九宫格和继任风险预警。

## 第六步：生成报告

点击右上角“生成报告”下拉按钮，选择“生成当前月报告”。

系统会根据当前 Excel 数据自动生成分析报告，内容包括总体情况、结构分析、绩效分析、风险提示、管理洞察和行动建议。

如果尚未导入 Excel，系统会生成明确标注为示例报告的预览内容。

## 第七步：导出文件

点击右上角“生成报告”下拉按钮，可以导出：

- PPT 报告。
- Word 报告。
- PDF 报告。
- Excel 报表。

如果还没有生成报告，系统会先自动生成当前月报告，再导出文件。

## 第八步：打印

点击右上角“打印”按钮，可以打印当前页面内容。

打印时系统会自动隐藏顶部按钮、Tab、导入控件和导出菜单，只保留适合打印的正文内容。

当前所在页面会影响打印内容：

- 在“数据看板”Tab 打印看板。
- 在“分析报告”Tab 打印报告。
- 在“数据录入”Tab 打印数据录入摘要。

## 缺字段时系统如何处理

Excel 缺少字段不会导致导入失败，系统不会编造假数据。

例如：

- 缺少 `绩效等级` 时，绩效图表会显示暂无数据。
- 缺少 `学历` 时，学历相关指标和图表会显示空状态。
- 缺少 `是否关键人才` 但存在 `人才分类` 时，系统会尽量根据可用字段判断。
- 缺少 `年龄` 但存在 `出生日期` 时，系统会尽量根据出生日期计算年龄。
- 缺少 `任职年限` 但存在 `入职日期` 时，系统会尽量根据入职日期计算任职年限。

分析报告和导出文件也会同步说明相关字段缺失导致哪些分析暂无法生成。
